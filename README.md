**Global Geopolitical Migration Mini-Project**

Hi! 👋
This is a data visualization project I made to explore how geopolitical risk connects to migration patterns around the world.
The goal of this project is to take raw data from an Excel file and turn it into clear visual graphs that help understand:
	Which countries have higher population risk
	Which countries have higher emigration pressure
	Where people are migrating to
	How countries are aligned politically
	What their strategic priorities are
	And how potential alliances connect through a network graph

🛠 What I Used
-Python
-Pandas (to handle data)
-Matplotlib (to create graphs)
-NetworkX (to build alliance networks)

📊 What This Project Does
The program loads an Excel dataset and gives you a menu where you can choose different visualizations.
You can:
	See a bar chart of population at risk
	See emigration pressure by country
	Find the most common migration destinations
	View bloc alignment distribution
	Analyze strategic priorities
	Compare risk vs emigration in a scatter plot
	Visualize alliances in a network graph
Each option generates a different type of graph.

🧠 Why I Made This
I wanted to practice:
	Data cleaning
	Data visualization
	Basic network analysis
	Writing interactive Python scripts
	It’s also a way to understand how instability and migration might be connected in a more visual and analytical way.

This project helped me combine coding with global analysis, and it’s part of my journey into data science and geopolitics.
Thanks for checking it out 🙂
