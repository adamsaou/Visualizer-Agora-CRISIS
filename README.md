I created this project to explore how geopolitical risk and migration patterns can be analyzed using data. I used Python to load an Excel dataset, clean and organize the information, and then turn it into different visualizations like bar charts, scatter plots, and even a network graph of alliances. The goal was to better understand how factors like population risk and emigration pressure might be connected, while also improving my skills in data analysis and visualization. I actually made this project for a Model United Nations conference, and it helped me win the Best Delegate award!
